# Data Engineering & Autonomous Agents Portfolio

Welcome to my centralized Data Engineering portfolio repository. This repository showcases a collection of production-grade data pipelines, professional take-home engineering assignments, and custom AI agent automation tools developed to optimize modern data workflows.

## 🚀 Repository Overview

This repository is organized into distinct functional sub-projects, reflecting expertise across the modern data stack, orchestration, cloud data warehousing, and AI integration:

*   **`cv_tailor`**: An LLM-powered orchestration pipeline built to automate resume optimization and tailored outreach. It leverages advanced prompt engineering principles to dynamically align core professional competencies with technical job specifications.
*   **`job-hunter-agent`**: An autonomous agent architecture designed to streamline, track, and optimize technical job acquisition workflows utilizing systemic logic and state management.
*   **`amiio-de-home-task`**: A comprehensive, end-to-end data engineering assignment demonstrating production-level coding standards, data modeling, and robust logic execution.
*   **`wix-de-home-assignment`**: A practical implementation focused on solving complex data modeling, transformation, and analytical engineering engineering challenges.

## 🛠️ Core Technical Stack

The projects housed within this repository heavily utilize the following technologies and methodologies:

*   **Languages:** Python (Pandas, PySpark), SQL
*   **Databases & Warehouses:** Google BigQuery, MySQL, NoSQL (MongoDB)
*   **Orchestration & Infrastructure:** Apache Airflow, Google Cloud Composer, GCP (GCS, BigQuery Ecosystem), Docker
*   **Data Engineering Practices:** Scalable ETL/ELT pipeline architecture, data validation, REST API integrations, performance optimization
*   **AI-Assisted Development:** Advanced Prompt Engineering, Autonomous Agent Workflows (Gemini, ChatGPT, Google Antigravity Agent Workspace)
